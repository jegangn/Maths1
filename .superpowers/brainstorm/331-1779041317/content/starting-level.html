Where does your son start today?

Synthesis sequences addition → subtraction → multiplication, but it assumes a kid who already counts and recognises numbers 1–10. Pick the level that matches him — Lesson 1 will start there.

A

Pre-number

Doesn't reliably name digits 1–10 yet, or can't count a small group of objects. Example: "Tap the picture that shows three dots."
Start here: subitising, counting, numeral recognition → then addition.

B

Comfortable counter

Knows numbers 1–10, can count objects, but hasn't done formal addition. Example: "How many?" with 4 ducks on screen.
Start here: jump straight into addition within 5 using tens-frames, then build up. Most common starting point for 5-year-olds.

C

Can add small numbers

Already does 2+3, 4+1 confidently on fingers. Example: "What is 3 + 4?"
Start here: addition within 10 → number bonds → subtraction.

D

Fluent up to 10

Adds within 10 instantly, ready for subtraction. Example: "7 − 3 = ?"
Start here: subtraction → multiplication as equal groups.

Tap a card, or describe his level in the terminal — even "he just turned 5, recognises some numbers but never done addition" is fine.
